# 栈

## 一、栈的应用

1. 编程语言中的函数调用
2. 操作系统中用户状态到内核态寄存器的保存
3. 网络消息的处理

## 二、简单栈特点

`先进后出`相当于如下操作

var arr = [a,b] =>  arr.pop() => 得到b（a先进入b后进入，但是可以先得到b）

## 四步分析法

1. 模拟：模拟题目的运行
2. 规律：尝试总结出题目的一般规律和特点
3. 匹配：找到符合这些特点的数据结构与算法
4. 边界：考虑特殊情况
